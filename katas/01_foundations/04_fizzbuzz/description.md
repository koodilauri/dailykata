Write a function `fizzBuzz` that takes a number `n` and returns a string:
- "Fizz" if divisible by 3
- "Buzz" if divisible by 5
- "FizzBuzz" if divisible by both
- The number as a string otherwise

Examples:
  fizzBuzz(3)   // "Fizz"
  fizzBuzz(5)   // "Buzz"
  fizzBuzz(15)  // "FizzBuzz"
  fizzBuzz(7)   // "7"
