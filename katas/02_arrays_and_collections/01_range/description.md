Write a function `range` that generates an inclusive array of numbers.

- `range(end)` — returns [0, 1, ..., end]
- `range(start, end)` — returns [start, ..., end]
- If start > end, count downward.

Examples:
  range(3)      // [0, 1, 2, 3]
  range(1, 4)   // [1, 2, 3, 4]
  range(5, 2)   // [5, 4, 3, 2]
  range(-2)     // [0, -1, -2]
