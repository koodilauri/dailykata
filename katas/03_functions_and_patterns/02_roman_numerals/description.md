Write a function `toRoman` that converts a positive integer to its Roman numeral representation.

Symbols: I=1, IV=4, V=5, IX=9, X=10, XL=40, L=50, XC=90, C=100, CD=400, D=500, CM=900, M=1000

Examples:
  toRoman(1)    // "I"
  toRoman(4)    // "IV"
  toRoman(9)    // "IX"
  toRoman(14)   // "XIV"
  toRoman(1994) // "MCMXCIV"
