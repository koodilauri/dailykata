Write a function `wordWrap` that inserts newlines into a string so that no line exceeds the given column width.

Examples:
  wordWrap('hello', 10)    // 'hello'
  wordWrap('abcdef', 3)    // 'abc\ndef'
  wordWrap('abcdefghi', 3) // 'abc\ndef\nghi'
  wordWrap('ab cd ef', 5)  // 'ab cd\nef'
  wordWrap('', 5)          // ''
