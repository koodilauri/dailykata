Write a function `isValid` that returns true if a string of brackets is correctly nested and closed.

Valid bracket pairs: `()`, `[]`, `{}`

Examples:
  isValid('()')       // true
  isValid('()[]{}')   // true
  isValid('([])')     // true
  isValid('(]')       // false
  isValid('([)]')     // false
  isValid('{[]}')     // true
