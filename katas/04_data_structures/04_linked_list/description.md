Implement a singly `LinkedList` class with the following methods:
- `append(value)` — add a value at the end
- `prepend(value)` — add a value at the start
- `delete(value)` — remove the first node with this value
- `find(value)` — return true if a node with this value exists
- `toArray()` — return all values as an array

The list should work with a simple `ListNode` that has `value` and `next` properties.
