Write a function `quicksort` that sorts an array of numbers in ascending order using the quicksort algorithm.

Examples:
  quicksort([3, 1, 4, 1, 5, 9])  // [1, 1, 3, 4, 5, 9]
  quicksort([])                   // []
  quicksort([1])                  // [1]
  quicksort([5, 4, 3, 2, 1])     // [1, 2, 3, 4, 5]
